每日美妝夾


為了摸索最適合的妝容，消費者常常得花費許多時間進行多方嘗試及查詢大量資料，在這個過程中可能有許多化妝品是因為過期而被丟棄，或是因為不適合而塵封在抽屜中。
我們希望所有人都能找到最適合他的妝容，因而設立了「Omni Folder 每日美妝夾」這個品牌 。「Omni」來自拉丁文的全能，也就是何時何地甚麼場景都能對應的美妝品牌。
藉由一系列一次性使用的化妝品，建立了獨有的美妝生態系統，減少變換與嘗試新妝容的壓力。搭配輕巧時尚的皮革美妝夾，讓每位女性都能在任何場合展現最自信美麗的自己。恣意遊走於白天上班時的專業、傍晚姊妺聚會的精緻以及是晚上派對的華麗等各種不同風格。

皮革, 紙, 鋁, 化妝品, 電鍍, SLA光敏樹脂, PP塑膠片
H67 x W110 x D27mm, H52 x W30 x D7mm, H46 x W9 x D6mm
