Class Corner — 遠距教學輔助設備


Class Corner 是一款遠距教學輔助設備，可以幫助國中小學生在上遠距課程時，有更好的互動性及體驗。此設計由智慧投影機、互動方塊與線上教學軟體組成，智慧投影機獨立連接網路，投影出即時課程內容，學生能更專注於課程、即時拍攝、分享作業畫面，幫助老師掌握學生學習成效。互動方塊透過搶答、提問等功能，增強遠距教學的互動性，也能透過加分、評價與私下通話功能，給予學生即時的反饋。
