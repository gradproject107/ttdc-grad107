Wrinkle Bubble


Wrinkle Bubble是一款以EPE(低密度發泡聚乙烯)為主軸進行材料製程研究所誕生的系列燈具。
EPE是常見的一次性緩衝包材，具有透光、熱縮、抗溶劑等特性。透過對材值得探索研究，研發出結合延伸、塑型、創造紋理的製造程序。
Wrinkle Bubble順應EPE的材料特性並激發其潛能，拓展其價值與製程上的可能性，並以獨一無二的樣貌重現工藝精神。

低密度發泡聚乙烯, 鐵, LED燈
吊燈 H800 x W800 x D600mm, 立燈 H1400 x W360 x D360mm, 壁燈 H600 x W900 x D100mm

TYPE 材料製程設計, 產品設計
DEPARTMENT 工業設計
